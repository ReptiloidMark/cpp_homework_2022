Вопрос №1
C++ - компилируемый язык программирования, что делает его намного быстрее интерпретируемых языков, поскольку компилятор "видит" весь код сразу (а не построчно),
вследствие чего может оптимизировать процесс компиляции. С++ поддерживает объекно-ориентированное программирование, что делает его удобным для написания целых, логически связанных, программ.
Вследствие его высокой производительности и удобства, он (язык) используется для разработки ПО, серверов и игр.


Вопрос №2
char (1 байт) - символьный тип данных, переменные этого типа хранят в себе символы ASCII (буквы, цифры, пробелы и прочие специальные символы).

string - строковый тип данных, в С++ задан как массыв char'ов. Переменные этого типа могут хранить в себе множество символов. Существует много функций, осуществляющих действия со строками:
	подсчет длинны, поиск элементов или подстрок, сложение строк и много другое.

int (4 байта) - целочисленный тип данных,  от −2 147 483 648 до 2 147 483 647. Для оптимищации памяти используют такие типы как short, unsigned short, long, unsigned long, long long, unsigned
long long.

float (4 байта) - численный тип данный с плавающей точкой. Представляет вещественное число ординарной точности с плавающей точкой в диапазоне +/- 3.4E-38 до 3.4E+38.

double (8 байт)  - численный тип данный с плавающей точкой. Представляет вещественное число (!)двойной точности с плавающей точкой в диапазоне +/- 1.7E-308 до 1.7E+308.

void - тип без значения.

Также следует упомянуть спецификатор auto, который автоматические определяет наиболее удобный тип данных.


Вопрос №3
Проблема переносимости возникает при необходитмости переноса переноса приложения с одной платформы на другую. Разработчики программ не всегда задумываются о том, чтоб их приложение нужно
будет в будущем перенести на другую платформу, поэтому используют особенности той, на которой пишут. Раньше для переноса кода существовали специальные программы, но их развитие происходило
намного медленнее, чем развитие платформ, вслдествие чего сейчас переносить приходится вручную.


Вопрос №4
Объявление переменной информирует компилятор о том, что где-то выделено n байт под хранение переменной такого-то типа с таким-то именем.

Определение переменной информирует компилятор о том, что память под переменную нужно взять в этом месте, где написано данное определение. Именно в этой единице трансляции.
Определение на всю программу может быть одно и только одно.

Присваивание - это задача значения уже существующей переменной.

Инициализация- это когда определение и объявление объединяют с присваиванием начального значения.


Вопрос №5
Выравнивание данных в памяти - это пропуск ячек памяти, для того, чтобы создать "shortcut" для обращения к ним машинного кода. Выравнивание увеличивает общий объем памяти, занимаемой данными,
но увеличивает производительность (скорость обращения при выполнении программы). Центральные процессоры в качестве основной единицы при работе с памятью используют машинное слово, размер 
которого может быть различным. Однако, размер слова всегда равен нескольким байтам. Как правило, машинное слово равно 2^k байтам.
